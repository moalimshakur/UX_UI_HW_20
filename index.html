EARTH TREK

Earth Trek is a mobile app that provides avid travelers a guide that allows them to experience the authentic culture of the host country

My Goal was to develop an app that provides a guide and a list of activities that allows avid travelers to fully experience the authentic culture of the host country

The Tools used during this project include Figma, Invision, Miro and Google Drive.

My Role:
  • UX Designer
  • Interviewing
  • Wireframing
  • Prototyping


RESEARCH

We planned and conducted 6 interviews with avid travelers to help identify the type of experiences that avid travelers desire, followed by definition and ideation stages
The interview insights were categorized and put through a feature prioritization matrix followed by an empathy map and user persona to further clarify user needs.
The result of previous steps was a well define value statement as follows: To serve as a guide to explore travel destinations by providing a list of activities, events and places that is ranked by locals, reviewed by other users and is authentic to the local culture.


IDEATION & USABILITY



We created a user flow based on the insights gained from the research stage in order to design a Lofi version of the mobile app. We then conducted a guerilla user test and ran the data collected through the 2X2 matrix which determined the features and changes that were high priority for both the designer and the end user




FINAL PROTOTYPE



Avid travelers have a sense of adventure and a deep desire to experience the world in its entirety. They want to experience the real culture without spending too much on one trip.
